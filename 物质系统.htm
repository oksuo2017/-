2 高值耗材, path: 我的资料\北大医疗信息技术有限公司\物质系统\高值耗材 (depth: 4)

--高值耗材关联价表
1、关联上bd_mm,bd_srv 
merge into  bd_mm a
  using  bd_srv b
  on (a.code=b.code and a.name=b.name)
  when matched then update set a.id_srv=b.id_srv where a.sd_mmtp in ('302')；
把bd_mm表关联上bd_srv  ，前台耗材器械基本信息只关联出来服务项目
